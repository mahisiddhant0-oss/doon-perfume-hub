# Doon Perfume Hub

A full-stack e-commerce and business management platform developed for a local perfume business. The application was designed to manage products, customers, orders, shipping, payments, and administrative operations from a centralized dashboard.

## Key Features

- Product creation, editing, deletion, pricing, stock, and variants
- Custom product and category creation
- Customer account and information management
- Standard and custom order creation
- Complete order management workflow
- Automated Delhivery AWB generation and shipment tracking
- Razorpay payment integration
- Product image and media management
- Administrative dashboard with protected access
- Customer enquiries and order history
- Responsive storefront for desktop and mobile devices

## Technology Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Zustand

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- Firebase Authentication
- JWT authentication

### Integrations
- Delhivery Shipping API
- Razorpay Payment Gateway
- Firebase Admin SDK
- SMTP email notifications
- Wix media storage

## Project Purpose

This project was built as a real-world solution for managing the online operations of Doon Perfume Hub, a local perfume business. It helped centralize product listings, customer information, custom orders, categories, payments, and shipping operations.

The original production website and cloud database are no longer active. This repository is preserved as a portfolio project demonstrating full-stack development, third-party API integration, authentication, database design, e-commerce workflows, and deployment experience.
